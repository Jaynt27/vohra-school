# 🎓 Vohra School Portal

A full-stack web application developed to simplify document management and communication within a school. The portal allows teachers to securely upload resumes, notes, and academic documents, while students and administrators can easily access and download them through a clean and responsive interface.

## 🚀 Features

- 🔐 Secure user authentication
- 👨‍🏫 Teacher dashboard for uploading resumes and academic documents
- 📄 Upload notes, PDFs, and study materials
- 👨‍🎓 Student dashboard to access shared resources
- 👨‍💼 Admin panel for managing uploaded documents
- 📥 Download documents anytime
- 📱 Responsive UI for desktop and mobile
- ⚡ Fast document retrieval using MongoDB

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Jaynt27/vohra-school.git
```

### Navigate to the backend

```bash
cd vohra-school/backend
```

### Install dependencies

```bash
npm install
### Start the server

```bash
npm start
```

The application will run at:

```
http://localhost:5000
```
